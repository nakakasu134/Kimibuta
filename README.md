## 作品概要：
　よさこいで用いられる打楽器「鳴子」を体験できるリズムゲーム。画面内の鳴子に向かって落ちて来るノーツの位置に合わせてタイミング良く鳴子を鳴らすことで、スコアを獲得する。
## 制作経緯・成果：
　私自身がよさこいに参加する中で、「踊りの経験の有無に関わらず誰でも始めることが出来る」「踊りを通じて人と人とが繋がり合い、一体感を得られる」という魅力を感じた一方で、「それらの魅力は実際に参加してみないと分かりづらい」「よさこい未経験者がよさこいを体験するための入り口が薄い」という課題を感じ、その解決のためにゲームという主体的な体験を提供出来るコンテンツでよさこいを発信することで、よさこいの魅力をより多くの人々に知ってもらえると考えた。
　本プロジェクトは、上記の展望を見据つつ短期間・少人数での制作でも成果を残し、将来的な課題解決実現に繋げるために「よさこいの要素の１つである『鳴子』を用いた簡易的なよさこい体験」の実装を目指し、展示会でフィードバックを得ることを目標として進行させた。立命館大学映像学部学生による展示イベント「EIZO JUNCTION × キャリアフェア」と同学主催の地域交流イベント「Global Week」でそれぞれ展示・体験フィードバックを行い、延べ49人によるアンケート回答を獲得した。
## 作品の特徴：
### ・鳴子を用いたデバイス
　デバイスはIMUによる回転速度とマイクの音の情報をアプリケーションに送信する。鳴子を鳴らした音と鳴子の回転を両方検知する方式で鳴子の判定をとることで、即時に正確な判定を実装することができた。
### ・ゲーム進行に伴う難易度・演出の変化
　ゲーム序盤のノーツの数を抑え、出現パターンも単調にしたことで初見でもプレイしやすくした。徐々に譜面を難しくしていき、同時にエフェクトも豪華にすることでプレイし飽きないゲーム進行を実現できた。
### ・物理的な音と連動したサウンドデザイン
　本編内のSEをノーツ判定の成功・失敗に絞り、成功判定も反響音のような薄い音にデザインするなど、デバイスに用いている鳴子本来の音を中心に据えたサウンドデザインを施した。展示会ではその音から観覧者の興味を惹きやすく、アンケート回答における体験の動機の半数以上が「他のプレイヤーがプレイしているのを見てやってみようと思った」という結果になった。
